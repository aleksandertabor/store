# store
## Install
- Docker is required to run this project

Run the following commands to set up the project:
- `make install`

## Usage
Open the browser and go to `http://localhost`

#### Login
- E-mail: `test@example.com`
- Password: `password`

or register a new user

## Start project
- `make up`
## Stop project
- `make down`
## Run tests
- `make test`
